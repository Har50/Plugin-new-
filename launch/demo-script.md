# 30-Second Demo Script

**Goal:** Show the problem → show the fix. Fast.

---

**0:00–0:05** — Split screen. Left: someone using OpenCode normally. Right: nothing.

> "Your AI coding agent does a lot. But do you know *what* it actually did?"

**0:05–0:15** — Screen shows the OpenAudit dashboard. Scroll through a session list.

> "OpenAudit logs every session, every tool call, every file change — automatically."

**0:15–0:25** — Click into a session. Show tool calls, file edits, errors.

> "See the exact commands the AI ran. Which files it touched. Errors it hit."

**0:25–0:30** — Click "Export Report." Show JSON download.

> "Export compliance-ready reports. Install in 30 seconds. Keep your CTO happy."

---

## Recording tips

- Use **terminal + browser** side by side
- Dark mode only (matches the product)
- No music. Voiceover or text overlay.
- Post on Twitter/X with caption: "Your AI coding agent has no audit trail. Fix that in 30s."
