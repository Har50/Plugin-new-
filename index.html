How it works

1

Install the plugin

Drop one file into .opencode/plugins/ or npm install @har53/opencode-audit

2

Use OpenCode normally

Every action is automatically logged to a local SQLite database

3

Open the dashboard

npm run dashboard → see everything at localhost:3456
